# TCPendant — Thermal Core Pendant

A compact, hand‑held and wearable thermal device built around an aluminum shell and a PI film heater. The goal is to create a small, safe, rechargeable heat source with smart temperature regulation.

## Purpose
- Portable and easy to carry
- Safe to touch
- Rechargeable over USB‑C
- Temperature‑regulated
- Small enough to wear as a pendant, hold in the hand, or keep in a pocket

## Status
Early development. Hardware prototyping in progress.
